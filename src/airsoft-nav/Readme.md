## Соглашения

- Папка приложения называется согласно энву THEME (здесь profitum)
- Иконки располагаются в папке `static/svg/<THEME>`
- Папку переводов i18n нельзя двигать
- Файл env.sample.jsonc нужен, контент должен совпадать с тем что будет использоваться для сборки
- Файл App.tsx нельзя двигать, он должен экспортировать компонент App

Всё остальное можно настраивать и двигать как угодно
