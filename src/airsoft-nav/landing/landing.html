✅ Без регистрации: Все данные локальны, серверная часть отсутствует.

Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности описывает, как владелец приложения «AirSoft Nav» (Елфимова Надежда Анатольевна) обрабатывает данные пользователей. Приложение не использует серверы для хранения или передачи персональных данных.

2. Какие данные хранятся в приложении

Только локально на вашем устройстве сохраняются:
• ID радиостанции (уникальный идентификатор DMR-устройства);
• Позывной (никнейм, введённый пользователем).

Мы НЕ собираем, НЕ передаём и НЕ храним:
• Геолокационные данные после завершения сессии;
• Контакты, фото, аудио, историю перемещений;
• Данные аккаунтов, email, телефон;
• Любую информацию на внешних серверах.

3. Как используются данные

• ID радиостанции и позывной используются исключительно для:
  – идентификации вас внутри команды на карте;
  – корректной работы обмена координатами через DMR-хотспоты;
  – работы деморежима внутри приложения.
• Все данные остаются в памяти вашего устройства и удаляются при удалении приложения.

4. Хранение и защита

Архитектура без сервера:
• Нет облачного хранения, нет баз данных, нет аналитики;
• Геопозиция передаётся только в реальном времени между устройствами команды через зашифрованный DMR-канал или Bluetooth;
• После выхода из приложения или завершения сессии координаты удаляются из оперативной памяти;
• ID радиостанции и позывной хранятся в защищённом хранилище устройства (SharedPreferences / Keychain).

5. Права пользователя

Вы можете в любой момент:
• Изменить или удалить свой позывной в настройках приложения;
• Очистить все локальные данные через «Настройки → Сброс»;
• Полностью удалить приложение — все данные будут безвозвратно удалены;
• Написать на airsoft@mailcase.net для консультации по обработке данных.
